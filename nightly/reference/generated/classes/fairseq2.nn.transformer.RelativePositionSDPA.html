Bases: SDPA

Computes scaled dot-product attention as described in Dai et al. [DYY+19].

Parameters:

  • model_dim (int) – The dimensionality of the model.

  • attn_dropout_p (float) – The dropout probability on attention weights.

Param:

num_heads: The number of attention heads.

Param:

pos_encoding: The relative positional encoding table.

  • queries (Tensor) – The queries. Shape: \((*,S,K)\), where \(*\) is any number of batch dimensions including none, \(S\) is the sequence length, and \(K\) is the key size.

  • keys (Tensor) – The keys. Shape: \((N,S_{kv},K)\), where \(*\) is any number of batch dimensions including none, \(S_{kv}\) is the key/value sequence length, and \(K\) is the key size.

  • values (Tensor) – The values. Shape: \((N,S_{kv},V)\), where \(*\) is any number of batch dimensions including none, \(S_{kv}\) is the key/value sequence length, and \(V\) is the value size.

  • mask (Tensor | None) – The float mask that will be added to the attention weights before computing the attention.
