SinusoidalPositionEncoder

final class fairseq2.nn.SinusoidalPositionEncoder(encoding_dim, max_seq_len, _legacy_pad_idx=None, device=None, dtype=None)[source]

Bases: PositionEncoder

Encodes sequences with fixed sinusoidal positional information.

The positional encodings are initialized as in tensor2tensor which differs slightly from the description in section 3.5 of Vaswani et al. [VSP+17]. This means instead of

\[ \begin{align}\begin{aligned}PE_{(pos, 2i)} = \text{sin}(pos/10000^{2i/d_{model}})\\PE_{(pos, 2i+1)} = \text{cos}(pos/10000^{2i/d_{model}})\end{aligned}\end{align} \]

we use

\[ \begin{align}\begin{aligned}PE_{(pos, i)} = \text{sin}(pos/10000^{i/d_{model}})\;\text{for}\;i\; <\frac{d_{model}}{2}\\PE_{(pos, i)} = \text{cos}(pos/10000^{i/d_{model}})\;\text{for}\;i\;\geq\frac{d_{model}}{2}\end{aligned}\end{align} \]

See here for more information.

Usage:

>>> import torch
>>>
>>> from fairseq2.nn.position_encoder import SinusoidalPositionEncoder
>>>
>>> m = SinusoidalPositionEncoder(encoding_dim=4, max_seq_len=16)
>>>
>>> seqs = torch.ones((3, 4))
>>>
>>> m(seqs)
tensor([[ 1.0000e+00,  1.0000e+00,  2.0000e+00,  2.0000e+00],  # pos 0
        [ 9.4147e-01,  2.0000e-04,  6.4030e-01,  2.0000e+00],  # pos 1
        [ 1.0930e-02,  3.0000e-04, -5.1615e-01,  2.0000e+00]]) # pos 2
Parameters:

  • encoding_dim (int) – The dimensionality of positional encodings.

  • max_seq_len (int | None) – The expected maximum sequence length.

forward(seqs, padding_mask=None, state_bag=None)
Parameters:

  • seqs (Tensor) – The sequences to encode with positional information. Shape: \((N,S,E)\), where \(N\) is the batch size, \(S\) is the sequence length, and \(E\) is the dimensionality of the positional encodings.

  • padding_mask (Tensor | None) – The float padding mask of seqs. Shape: \((N_{msk},S)\), where \(N_{msk}\) is the mask batch size and \(S\) is the sequence length. \(N\) can be a multiple of \(N_{msk}\) in which case the mask will be tiled before being applied.

  • state_bag (IncrementalStateBag | None) – The state bag to use for incremental evaluation.

Returns:

The input sequences with positional information encoded. Shape: Same as seqs.

Return type:

Tensor
