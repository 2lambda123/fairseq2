nn.Linear

Applies a linear transformation to incoming data using weights and bias.

Generates a causal attention mask for self attention.

nn.transformer.RelativePositionSDPA

Computes scaled dot-product attention as described in Dai et al. [DYY+19].

nn.transformer.StandardFeedForwardNetwork

Represents a Transformer feed-forward network as described in Vaswani et al. [VSP+17].

nn.transformer.StandardMultiheadAttention

Represents a Transformer multi-head attention as described in Vaswani et al. [VSP+17].

nn.transformer.StandardTransformerDecoder

Represents a Transformer decoder as described in Vaswani et al. [VSP+17].

nn.transformer.StandardTransformerDecoderLayer

Represents a Transformer decoder layer as described in Vaswani et al. [VSP+17].

nn.transformer.StandardTransformerEncoder

Represents a Transformer encoder as described in Vaswani et al. [VSP+17].

nn.transformer.StandardTransformerEncoderLayer

Represents a Transformer encoder layer as described in Vaswani et al. [VSP+17].

nn.transformer.StoreAttentionWeights

optim.lr_scheduler.CosineAnnealingLR

Represents the abstract base class for learning rate schedulers.

optim.lr_scheduler.MyleLR

Represents a scaled version of NoamLR that preserves the base learning rate of the associated optimizer.

optim.lr_scheduler.NoamLR

Represents the learning rate schedule described in Section 5.3 of Vaswani et al. [VSP+17].

optim.lr_scheduler.PolynomialDecayLR
